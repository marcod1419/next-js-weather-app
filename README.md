# Sample Weather App

This is a sample weather app that uses the OpenWeatherMap API to get the current
weather for a given city.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Visit `http://localhost:3000`

## Tech Stack

- React
- Next.js
- CSS Modules
- Jest
- React Testing Library
- OpenWeatherMap API
- Vercel
